Summary

In a nut shell, it works like this:

  • Mobile app button is clicked, and RESTful call is made to Electric Imp Agent (ie. http://agent.electricimp.com/RbNoShy8EqMC).
  • Agent receives http request and calls event handler, which then passes arguments to Device (the physical chip)
  • Device calls Event Handler to change IO pins from high to low (or vise versa depending on wiring setup)
  • Device sends agent a success or error message
  • Agent sends back response to mobile app
  • Mobile app adjusts angular model through '$powerbar' service

    Developer Docs

    Electric Imp is a cloud based IoT platform/device that is really easy to use and setup. This is the link between your mobile app and your physical device. Their patented 'Blink Up' technology means you can have an Imp powered up and connected to the web in < 2mins.

    There are two code bases to maintain for Electric Imp, one for the device and one in their cloud. Coding them is super easy, very much like JS, and their IDE is really easy to use.
